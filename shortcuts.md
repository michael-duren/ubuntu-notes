# Ubuntu Linux Keyboard Shortcuts

## Desktop / Workspace Navigation

| Shortcut | Action |
|---|---|
| `Super` | Open Activities overview |
| `Super + Page Up` | Move to workspace above |
| `Super + Page Down` | Move to workspace below |
| `Ctrl + Alt + ↑` | Move to workspace above (alt binding) |
| `Ctrl + Alt + ↓` | Move to workspace below (alt binding) |
| `Super + Shift + Page Up` | Move current window to workspace above |
| `Super + Shift + Page Down` | Move current window to workspace below |
| `Super + Home` | Jump to first workspace |
| `Super + End` | Jump to last workspace |

## Window Management

| Shortcut | Action |
|---|---|
| `Super + ←` | Snap window to left half |
| `Super + →` | Snap window to right half |
| `Super + ↑` | Maximize window |
| `Super + ↓` | Restore / unmaximize window |
| `Super + H` | Minimize (hide) window |
| `Super + Shift + ←/→` | Move window to left/right monitor |
| `Alt + F4` | Close window |
| `Alt + F7` | Move window (use arrow keys or mouse) |
| `Alt + F8` | Resize window (use arrow keys or mouse) |
| `Alt + F9` | Minimize window |
| `Alt + F10` | Toggle maximize |
| `Alt + Space` | Window menu |

## Window Switching

| Shortcut | Action |
|---|---|
| `Alt + Tab` | Switch between windows |
| `Alt + Shift + Tab` | Switch windows (reverse) |
| `Super + Tab` | Switch between applications |
| `Alt + `` ` (backtick) | Switch between windows of the same app |

## General / System

| Shortcut | Action |
|---|---|
| `Super + L` | Lock screen |
| `Super + D` | Show desktop (minimize all) |
| `Super + A` | Open application grid |
| `Super + S` | Quick settings / calendar panel |
| `Ctrl + Alt + T` | Open terminal |
| `Ctrl + Alt + Delete` | Log out |
| `Print Screen` | Screenshot tool |
| `Shift + Print Screen` | Screenshot area selection |
| `Alt + F2` | Run command dialog |

---

*Based on GNOME desktop (Ubuntu 22.04+). Some bindings may vary depending on your desktop environment or custom keybindings.*
